Holo x86 · κ-block disk — a real x86 Linux whose disk is a content-addressed κ-DAG
The disk is NOT one blob. It is an ordered set of κ-blocks; v86 reads it lazily and a service worker resolves each block by its content address and re-derives it before the guest sees a byte (Law L5). Only blocks the guest touches are ever fetched.
