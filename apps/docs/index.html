Holo Docs

Serverless · end-to-end encrypted · real-time co-editing. A saved document is its content address.

Open document

Paste a document content address (holo://sha256:… or a κ). Locally saved documents open instantly; a shared link reconstructs the document live from the room and verifies it re-derives to its κ.

Export

Portable, in-browser exports. The sealed .holo snapshot is the content-addressed source of truth.

Settings

User Data (Tools ▸ Options ▸ User Data) and sharing mode (LibreOffice “Share Document”).

Off shared mode: everyone edits live (Calc multi-user model). The lock is advisory, like LibreOffice’s file lock.

Resolve Conflicts

The same content was changed concurrently. Choose which edit to keep (LibreOffice Calc shared-mode resolution).
