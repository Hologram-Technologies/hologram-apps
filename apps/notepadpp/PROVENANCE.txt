Holo Notepad++ — provenance and third-party attribution
=======================================================

The editor engine is CodeMirror 6, vendored verbatim (no hand-written editor code).
It is bundled offline from the upstream npm packages and served from the app itself
(Law L4: no CDN, no network). The host shell (index.html, notepad.js) is integration
glue: it composes CodeMirror's published extensions into a tabbed, Notepad++-style
editor and binds the toolbar, tabs and file I/O.

Note on the name
----------------
"Notepad++" is the name of the native Windows editor by Don Ho. Holo Notepad++ is an
independent, browser-native editor in the same spirit (fast, lightweight, tabbed,
syntax-highlighted); it does not include or derive from the Notepad++ source code,
which is a native Win32/Scintilla application and has no web build.
